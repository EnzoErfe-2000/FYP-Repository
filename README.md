# Running the application locally

1. Open a Terminal
• Install requirements
  ..\fyp-nlp-reviews> pip install -r requirements.txt
• Navigate to the project directory.
  ..\fyp-nlp-reviews> cd .\dashboard\

3. Activate the Virtual Environment (Optional)
• Windows:
  venv\Scripts\activate
• Linux/Mac:
  source venv/bin/activate

5. Run the Application
• streamlit run app.py
  ..\fyp-nlp-reviews\dashboard> streamlit run.app.py

5. Accessing the Dashboard
• Once the application starts, a web browser window will automatically open.
• If not, access: http://localhost:8501
